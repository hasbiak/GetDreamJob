## About The Project

GDJ (Get Dream Job) is a website that connects workers with recruiters, in order to make it easier for recruiters to find, sort, and get the best talents with the skills their companies need.

## Built With

[![Vue](https://img.shields.io/badge/Vue-v2.6.11-green)](https://github.com/vuejs/vue)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-v4.5.x-blue)](https://github.com/bootstrap-vue/bootstrap-vue)

## Requirements

1. <a href="https://nodejs.org/en/download/">Node Js</a>
2. Node_modules `npm install` or `yarn install`
3. Backend API GDJ-Back-End [`here`](https://github.com/Cotllinz/GDJ-Back-End)

## Getting Started

1. Download this Project or you can type `git clone https://github.com/Cotllinz/GDJ-Front-End`
2. Open app's directory in CMD or Terminal
3. Type `npm install` or `yarn install`
4. Add .env file at root folder project

```
VUE_APP_URL=localhost:3000
```

5. Type `npm run serve`
